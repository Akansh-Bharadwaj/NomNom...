# NomNom...
A simple game I made in java for a school project in 2017

# Installation/Usage

- download / clone this repository in a suitable directory.
```bash
$ git clone https://github.com/AkitoshTheBold/NomNom....git
```
- run as 
```bash
$ java -jar ./NomNom.jar
```
- Enjoy!!
